3 уровень - сетевой уровень (маршрутизация и IP адреса) ( dns запрос - ип ответ - подставляем в пакет) [маршрутизатор]
4 уровень - транспортный ( обеспечивает передачу данных по сети)
TCP - Transmission control protacol - контроль за передачей данных
UDP - User Datagram protocol  - могут частично потеряться
///////// остальные уровни для разработки софна /////////
5уровень - сиансовый ( в зуме видеоконференция выбирается каким Кодеком будет кодирываться сигнал и
Этот кодек должен быть у двоих собеседников)
6уровень - уровень представления ( кодирование, сжатие) [ приходит фотка в ТГ но она приходит как 0101011101 а потом преобразется в формат jpg)
7 уровень - уровень приложений (http/ https) - приложения имели доступ к файлам , электронной почте и запросы в БД

TLS(Transport Layer Security - более крутая версия ССЛ
